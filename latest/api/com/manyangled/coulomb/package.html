  6. trait BaseUnit extends UnitExpr

    Base units are axiomatic units.

    Base units are axiomatic units. The Standard International (SI) unit system defines seven Base units, which coulomb defines in SIBaseUnits. Exactly one BaseUnit is defined for each category of quantity (e.g. length, mass, time, etc).
